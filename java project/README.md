# Vehicle Rental System

A Java-based vehicle rental system demonstrating OOP concepts including Encapsulation, Inheritance, and Polymorphism.

## How to Run

### Step 1: Compile the Java Files
Open terminal/command prompt in the project directory and run:
```bash
javac Vehicle.java Car.java Motorcycle.java Truck.java RentalSystem.java
```

### Step 2: Run the Program
```bash
java RentalSystem
```

## Project Structure

- **Vehicle.java** - Base class (Parent class)
- **Car.java** - Subclass extending Vehicle
- **Motorcycle.java** - Subclass extending Vehicle
- **Truck.java** - Subclass extending Vehicle
- **RentalSystem.java** - Main class with menu-driven interface

## Features

1. Display available vehicles
2. Rent a vehicle (with optional discount)
3. Return a vehicle
4. Exit the system

## OOP Concepts Demonstrated

- ✅ **Encapsulation** - Private fields with getters/setters
- ✅ **Inheritance** - Car, Motorcycle, Truck extend Vehicle
- ✅ **Polymorphism** - Method overriding and method overloading
- ✅ **Abstraction** - Abstracted vehicle operations
